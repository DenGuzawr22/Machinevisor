  • SUB update

    Allows to receive updates of machines that was subscribed through a publish msg to the machines/subscribe channel

    Accepts the following message:

    MachineUpdate
    object allOf
    restricted any
    machine_id
    required
    integer
    state
    required
    string
      Allowed values:
    • "OFFLINE"
    • "OFF"
    • "ON"
    • "ALLARM"
    timestamp
    required
    string
    format: date-time
    working_time
    required
    integer

    Time in minutes passed from the turn on of machine

    temperature
    required
    number
    format: flaotdecimal places <= 1

    temperature in celsius

    kWatt
    required
    number
    format: floatdecimal places <= 2
    machine_oil_level
    number
    format: float[ 0 .. 100 ]decimal places <= 2

    Remaining machine oil level in %

    array<string>

    Optional. Shows what values of machine provocate allarm state

    Items:
    string
      Allowed values:
    • "temperature"
    • "kWatt"
    • "machine_oil_level"
    restricted any
    uid: Modality
    modality
    required
    string
      Allowed values:
    • "SLEEP_MODE"
    • "ENERGY_ECONOMY_PRODUCTION_MODE"
    • "PRODUCTION_MODE"
    • "NO_MODE"

    Additional properties are allowed.

    Examples

    {
  "machine_id": 1,
  "state": "ALLARM",
  "timestamp": "2022-06-21T10:19:47.836Z",
  "working_time": 15,
  "temperature": 20.2,
  "kWatt": 5.23,
  "machine_oil_level": 100,
  "allarm": [
    "temperature"
  ],
